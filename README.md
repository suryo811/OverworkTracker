# ⏱️ MenuBar Time Tracker

A minimal macOS menu bar app to track how you spend your time.

![Preview](./tracker.png)

## ✨ Features
- Real-time app usage tracking  
- Daily totals & insights  
- Clean, native macOS UI  
- 100% local (privacy-first)

## 🚀 Why
Understand your time before trying to optimize it.
